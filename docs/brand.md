## 1. Ядро бренда

| Параметр | Значення |
|----------|----------|
| **Назва** | Каспер. Крок за кроком. |
| **Суть** | AI-тренер для поступового повернення у форму |
| **Слоган** | Не рекорд. Ритм, що тримає. |
| **Характер** | Спокійний, уважний, мотивуючий |
| **Образ** | Добрий розумний супутник поруч |
| **Емоція** | Затишок, довіра, турбота, стабільність |
| **Функція** | Щоденник активності + м'який тренер + контроль ритму |
| **Обіцянка** | Рухатись регулярно, без перевантаження і хаосу |

**Формула бренда:** Каспер — це теплий AI-супутник, який допомагає людині повернути рух і форму через маленькі стабільні дії: без тиску, без хаосу, крок за кроком.

---

## 2. Позиціонування і межі

| | |
|-|-|
| ✅ **Що це є** | AI-супутник і щоденник активності для поступового повернення у форму |
| 🚫 **Що це не є** | Не медичний консультант, не агресивний тренер, не застосунок для рекордів будь-якою ціною |
| 🎯 **Для чого** | Щоб тримати ритм, бачити прогрес, не перевантажуватись і рухатись системно |

**Головний сенс:**
- Не змусити людину стати спортсменом за тиждень, а повернути регулярний рух як нормальну частину дня
- Не створювати провину за пропуски, а м'яко повертати до ритму
- Не гнатися за рекордами, а будувати систему: ходьба, самопочуття, контроль навантаження, поступовість
- Показувати прогрес так, щоб людина бачила результат і хотіла продовжувати

## 3. Образ Каспера

| Символ | Роль |
|--------|------|
| 🐈 **Рудий кіт** | Основний емоційний образ бренда: живий, добрий, домашній, впізнаваний |
| 🐾 **Лапка** | Фірмовий знак маленького кроку, присутності й підтримки |
| 👣 **Сліди** | Метафора шляху: крок за кроком, без ривків, але послідовно |
| 🔊 **Муркотіння** | Майбутній аудіо-мотив спокою, підтримки й завершення дії |

**Як не зіпсувати образ:**
- Не робити Каспера мультяшно-дитячим або надмірно солодким
- Не перетворювати бренд на суто «котячу тему» — мотиви мають працювати на сенс руху
- Не використовувати агресивну спортивну естетику, крик, тиск, сором
- Базовий варіант — реалістичний буклетний стиль без зміни грудки кота
